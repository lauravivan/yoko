# yoko!

## Description

This is a website made with React.js to keep on track of important events and current actions.

## Features

- [x] Filter
- [x] Sort
  - [x] By day
  - [x] By color
  - [x] By alphabet
- [x] View
  - [x] Grid
  - [x] List
- [x] Themes
  - [x] Light
  - [x] Dark
- [x] Search
